# artefacts/ — Demo project

This directory would hold raw artefacts referenced by the journal: prompts, code snippets, generated outputs, schemas. For the demo we have left it empty. In a real project you would expect to find files here with frontmatter and a naming convention of `YYYY-MM-DD-slug.ext`.
